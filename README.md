# Claude Code Skills

Claude Code에서 사용하는 커스텀 스킬 모음.

## 스킬 목록

| 스킬 | 설명 |
|------|------|
| [system-doc](./system-doc/) | 팀 내부 시스템을 흐름/아키텍처/사용법 문서로 만든다 |
| [research-quality](./research-quality/) | 구조화된 리서치 보고서를 작성한다 |

## 설치

스킬 디렉토리를 `~/.claude/skills/`에 복사하면 된다.

```bash
cp -r system-doc ~/.claude/skills/
cp -r research-quality ~/.claude/skills/
```
